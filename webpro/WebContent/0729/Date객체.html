1. Date객체 생성

- new Date()
- new Date(123456788) : 1970.1.1일 기준으로 경과한 날짜
- new Date("1980.10.11")
- new Date(2000,01,23,12,30,12) : 년,월,일,시,분,초
- 월은 0월부터 시작, 숫자로 Date 객체 생성시 하나 작은 값으로 설정 한다.



2. Date객체 생성

- now = new Date()
- now.toLocaleString() : 날짜와 시간
- now.toLocaleDateString() : 날짜
- now.toLocaleTimeString() : 시간



3. 요일 구하기

- now = new Date()
- day = now.getDay() : 요일이 아니라 index값으로 나온다 0(일요일) ~ 6(토요일)
